# CS50W Project 1 – Wiki

This project is a Wikipedia-style encyclopedia web application built as part of
CS50's Web Programming with Python and JavaScript course.

The application allows users to view, search, create, and edit encyclopedia entries.

## Features

- View encyclopedia entries written in Markdown
- Search for entries by title
- Create new encyclopedia pages
- Edit existing pages
- Random page generator
- Markdown content automatically converted to HTML

## Technologies Used

- Python
- Django
- HTML
- CSS
- Markdown

## Project Source

This project was completed as Project 1 of CS50's Web Programming with Python and JavaScript course.
